Tentang Saya : Saya memiliki latar belakang dan keterampilan yang mencakup beberapa bidang teknologi dan produktivitas. Dengan pengalaman di web programming, saya mampu merancang, membangun, dan mengelola situs web atau aplikasi berbasis web. Saya juga menguasai penggunaan perangkat lunak office seperti Microsoft Word, Excel, dan PowerPoint, yang mendukung aktivitas administratif, pengolahan data, dan presentasi profesional.

Dengan kombinasi keterampilan ini, saya memiliki fleksibilitas dan kemampuan untuk menjalankan berbagai peran di bidang teknologi informasi dan administrasi.

Sistem Informasi Diving di Kabupaten Bolaang Mongondow Selatan

Sistem informasi ini berbasis web dan berfungsi untuk mempermudah wisatawan, operator selam, dan pengelola lokasi dalam mengakses informasi terkait lokasi penyelaman, fasilitas, dan panduan wisata selam.

Aplikasi Visit Gorontalo

Aplikasi Visit Gorontalo adalah platform berbasis mobile atau web yang dirancang untuk mempromosikan destinasi wisata di Provinsi Gorontalo. Aplikasi ini bertujuan untuk memberikan informasi lengkap mengenai tempat wisata, budaya, kuliner, dan aktivitas menarik yang dapat dinikmati oleh wisatawan lokal maupun internasional. Dengan menggunakan aplikasi ini, pengguna dapat merencanakan perjalanan mereka dengan lebih mudah, serta mengeksplorasi keindahan Gorontalo secara lebih interaktif.
